# Ratio Cropper

一个零依赖的本地图片裁剪小程序，支持：

- 上传本地图像并在页面中可视化预览
- 选择预设裁剪比例
- 自定义宽高比
- 拖动裁剪框移动位置
- 拖动四角调整裁剪范围
- 导出 PNG 或 JPEG

## 使用方式

直接用浏览器打开 [index.html](./index.html) 即可。

## 交互说明

- 裁剪框内部：拖动移动
- 裁剪框四角：拖动缩放
- `自由裁剪`：宽高可独立调整
- `自定义`：按输入的宽高比锁定裁剪框

## 文件结构

- `index.html`：界面结构
- `styles.css`：页面样式
- `script.js`：裁剪逻辑和导出逻辑
